# Sistema Bancário com Python

Este é um sistema bancário em Python que simula operações básicas de um banco, como depósitos, saques e exibição de extrato. O sistema também controla o saldo da conta, o número de saques realizados e possui um limite de saques por sessão.

## Funcionalidades

- **Depositar:** Adicione fundos à sua conta bancária.
- **Sacar:** Retire fundos da sua conta, respeitando o saldo disponível e o limite estabelecido.
- **Extrato:** Visualize o extrato da sua conta, listando todas as transações realizadas e o saldo atual.
- **Sair:** Encerre a sessão bancária com segurança.

## Utilização

Para utilizar o sistema bancário, execute o arquivo `sistema_bancario.py`. Siga as instruções apresentadas no menu interativo para realizar operações como depósitos, saques e consulta do extrato da conta.

Certifique-se de inserir valores válidos e siga as orientações fornecidas durante a interação com o sistema.

## Requisitos

- Python 3.x
